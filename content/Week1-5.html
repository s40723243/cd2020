Week1-5

Week1

建立個人的cd2020倉儲

1.下載2019Fall可攜套件.7z 打開舊的start

2.登入自己的github，建立一個cd2020的新倉儲

3.輸入 git clone  https://github.com/s40723243/cd2020 

4.輸入git submodule add https://github.com/mdecourse/cmsimde.git 

5.將up_dir的裡的資料複製到cd2020

6.打開近端開始編譯自己的倉儲

Week2

python 3.7.3系統更新至python 3.8.2

1.新增一個資料夾cd2020，在cd2020下新建資料夾data，在data下新增資料夾py382及wscite432、portablegit

2.進入Python，點選Downloads下的Windows，並點選Latest Python 3 Release - Python 3.8.2

3.下載Windows x86-64 executable installer

4.開始下載，將pip選項取消，接著按下next並將檔案下載至py382。

5.下載SciTE系統 full 64-bit download，解壓縮至data下的資料夾wscite432。

 6.建立新的start.bat及stop.bat

7.下載MSYS2 msys2-x86_64-20190524.exe 至data下。

8.下載PortableGit 64-bit Git for Windows Portable 至 portablegit的資料夾下。

9.安裝pip，到https://bootstrap.pypa.io/get-pip.py頁面，另存新檔到data下。
